CUSTOMER2AI MAINTENANCE MODE PLUGIN
Quick Installation Guide
=====================================

STEP 1: UPLOAD THE PLUGIN
--------------------------
Upload the entire "customer2ai-maintenance-mode" folder to:
/wp-content/plugins/

Your file structure should look like this:
/wp-content/plugins/customer2ai-maintenance-mode/
    ├── customer2ai-maintenance-mode.php
    ├── assets/
    │   └── robot.webp
    └── README.md


STEP 2: ACTIVATE THE PLUGIN
----------------------------
1. Log into your WordPress admin panel
2. Go to: Plugins → Installed Plugins
3. Find "Customer2AI Maintenance Mode"
4. Click "Activate"


STEP 3: CONFIGURE SETTINGS
---------------------------
1. Go to: Settings → Maintenance Mode
2. Check "Enable Maintenance Mode"
3. Customize the title and message (optional)
4. Add your social media URLs (optional)
5. Click "Save Changes"


STEP 4: TEST THE PLUGIN
------------------------
1. Open your website in an incognito/private browser window
2. You should see the maintenance page
3. As an admin, you can still access the site normally


IMPORTANT NOTES:
----------------
✓ Administrators can always access the site
✓ The maintenance page shows a 503 HTTP status (good for SEO)
✓ To disable maintenance mode, uncheck "Enable Maintenance Mode" in settings
✓ The plugin matches the exact design from your PDF file
